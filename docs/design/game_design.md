# ゲームデザイン

## ゲームの概要

- 多くのモンスターにスムーズに交差点を渡らせるゲーム。
- タワーディフェンスゲームと交通整理ゲームの特徴を掛け合わせている。

## ゲームのルール

- ゲームプレイの中で、モンスターの往来を捌くフェーズと交差点のリソースを整備するフェーズを交互にプレイする。
- 各プレイの開始時にはリソースは少なく、フェーズを重ねると強化されていくローグライク的なプレイ感。

(以下、wip)

- モンスターの往来フェーズでは、お金に相当する「魂」を管理する。
- モンスターが無事に交差点を渡り切ると魂をゲットできる。
- ゲットした魂をアイテムに変換する。アイテムには交差点に配置するものや、場やモンスターに直接効果をもたらすものがある。
- モンスターの中には、交差点を渡らせるのが難しい「ボス」のようなモンスターがいる。
  - それ自体の進みが遅く加速やルート指示が必要なものや、他のモンスターの通行を邪魔する障害物を配置するもの
  - 弱いモンスターを渡しつつ、得られた魂を使ってボスを処理する、というゲームプレイを想定している。

## 往来フェーズのゲームルール

- フィールドはマス目で区切られている。レベルごとに差異があって良いが、大体 10x10 くらいのサイズのイメージ
- 配置アイテムはこのマス目に配置される。
- モンスターは正確にマス目の上を移動する訳では無いが、基本的にはマス目に沿って移動してくる。
- フィールドは四角形で、モンスターの往来は上下左右の4方向。(2-3方向にしても良いかも？)

### アイテムと戦略について

- 加速床
  - 時間制限があるキャラクターをスムーズにゴールさせる。
- ジャンプ台
  - 1マス飛び越える。タテとヨコが交差する場合有効。
  - ジャンプしているモンスター同士は衝突する

### 障害物について

- ウェーブ開始時に配置される。
- 泥沼？
  - 遅くなる
  - 破壊は大変。回り道を作る。

### モンスターについて

妖怪のイメージなので、モンスターも妖怪。

- ゴースト
  - 他のモンスターに当たらずすり抜ける。簡単。
- (隊列を組むキャラクター)
  - モンスターが縦に連続することで、「交差によるリスク」が大きくなりプレイヤーの対処が求められる。

## 準備フェーズのゲームルール

- 往来フェーズにも配置はできるので、準備フェーズに大きく改築することは想定していない。
- ウェーブごとに3つくらいのアイテムから選択できる定番の形式のイメージ
