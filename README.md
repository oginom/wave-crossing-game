# wave-crossing-game

This is an entry for [GitHub Game Off 2025](https://itch.io/jam/game-off-2025), GitHub's annual month-long game jam.

## About Game Off 2025

**Theme:** WAVES

**Duration:** November 1, 2025 - December 1, 2025

Game Off is GitHub's annual game jam now in its 13th year. Participants can interpret the theme "WAVES" in various ways - from physical waves on water, to repeating patterns of motion/energy/sound, or even gestures of greeting.
